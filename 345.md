**Level:** Easy
**Problem:**
Given a string s, reverse only all the vowels in the string and return it.

The vowels are 'a', 'e', 'i', 'o', and 'u', and they can appear in both lower and upper cases, more than once.

**Example 1:**

Input: s = "IceCreAm"
Output: "AceCreIm"
Explanation:
The vowels in s are ['I', 'e', 'e', 'A']. On reversing the vowels, s becomes "AceCreIm".

**Example 2:**

Input: s = "leetcode"
Output: "leotcede"

**Constraints:**

1 <= s.length <= 3 * 105
s consist of printable ASCII characters.

**Solution:(python3)**

   class Solution:
    def reverseVowels(self, s: str) -> str:
        l=0
        r=len(s)-1
        a=list(s)
        vowel="aeiouAEIOU"
        while l<r:
            if a[l] in vowel and a[r] in vowel:
                a[l],a[r]=a[r],a[l]
                l+=1
                r-=1
            elif a[l] not in vowel:
                l+=1
            else:
                r-=1
        return ''.join(a)
